# What is JavaScript?

JavaScript is a high-level, interpreted programming language used to create interactive and dynamic web applications.

## Real-Life Analogy

Think of a car:

- HTML → Car Structure
- CSS → Car Paint & Design
- JavaScript → Engine

Without JavaScript, a website can look beautiful but cannot interact with users.

---

## Why JavaScript Matters

JavaScript powers:

- Websites
- Mobile Apps
- Desktop Applications
- Backend APIs
- AI Dashboards
- Games

Today, JavaScript is one of the most widely used programming languages in the world.

# JavaScript Engines

## What is a JavaScript Engine?

A JavaScript Engine converts JavaScript code into machine code that computers can understand.

## Popular Engines

| Engine | Browser |
|----------|----------|
| V8 | Chrome |
| SpiderMonkey | Firefox |
| JavaScriptCore | Safari |
| Chakra | Legacy Edge |

# Single Thread vs Multi Thread

JavaScript executes one task at a time.

```javascript
console.log("Task 1");
console.log("Task 2");
console.log("Task 3");
```

---

# Ways to Add JavaScript

## Inline JavaScript

```html
<button onclick="alert('Hello')">Click Me</button>
```

## Internal JavaScript

```html
<script>
  alert("Hello");
</script>
```

## External JavaScript

```html
<script src="app.js"></script>
```

---

# Async vs Defer

## Async

```html
<script async src="app.js"></script>
```

## Defer

```html
<script defer src="app.js"></script>
```
